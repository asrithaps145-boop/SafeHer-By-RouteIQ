# SafeHer By Synergy Seekers

SafeHer is a women-centric safety navigation platform developed by Team Synergy Sykers for the Elevate Hackathon.

## Problem
Women often face safety risks while traveling, particularly during late hours and in unfamiliar locations. Traditional navigation applications prioritize speed and distance rather than user safety.

## Solution
SafeHer provides safety-oriented route recommendations, emergency assistance features, and trusted contact alerts to help users travel more confidently.

## Features
- Safe Route Recommendations
- Emergency SOS Alerts
- Trusted Contact Notifications
- Route Safety Scores
- User-Friendly Dashboard
